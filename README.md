# 🎮 经典小游戏合集

一个 C++ 编写的经典控制台小游戏合集，包含贪吃蛇和飞机大战两款游戏，并配有统一的游戏选择菜单。

---

## 📋 项目结构

```
e:\学习资料\tanchishe\
├── menu.cpp / menu.exe          ← 游戏选择菜单（入口程序）
├── hello.cpp / hello.exe        ← 贪吃蛇 (Snake Game)
├── airplane_war.cpp / .exe      ← 飞机大战 (Airplane War)
└── README.md                    ← 本文件
```

---

## 🐍 贪吃蛇 (Snake Game)

**文件：** `hello.cpp`

经典贪吃蛇游戏，玩家控制一条蛇在围墙内移动并吃掉食物来增长身体和得分。

### 操作方式
| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `X` | 退出游戏 |
| `R` | 游戏结束后重新开始 |

### 游戏规则
- 蛇头碰到墙壁或自己的身体则游戏结束
- 每吃到一个食物（`@`）得10分，蛇身增长一节
- 游戏区域大小为 20×20

## ✈️ 飞机大战 (Airplane War)

**文件：** `airplane_war.cpp`

控制飞机发射子弹击落敌机，躲避敌机及其发射的子弹。

### 操作方式
| 按键 | 功能 |
|------|------|
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格键` | 发射子弹 |
| `X` | 退出游戏 |
| `R` | 游戏结束后重新开始 |

### 游戏规则
- 玩家操控飞机（`^`）左右移动
- 按空格发射子弹（`|`）击落敌机（`V` / `W` / `M`），每架得10分
- 需躲避敌机及其发射的子弹（`*`）
- 共3条命，被击中或敌机到达底部均会扣命
- 每击落5架敌机升级，敌机生成加快、射击频率提高
- 游戏区域大小为 30×25

## 🚀 游戏选择菜单 (Menu)

**文件：** `menu.cpp`

统一的游戏启动入口，让玩家在菜单中自由选择要玩的游戏。

### 操作方式
| 按键 | 功能 |
|------|------|
| `↑` / `↓` | 移动选择光标 |
| `Enter` | 确认选择并启动游戏 |
| `ESC` | 退出菜单 |

### 功能特点
- 美观的控制台 UI，带边框和彩色文字
- 选中项高亮显示，方向键循环选择
- 游戏结束后自动返回菜单，方便切换
- 退出时显示友好的告别提示

## ⚙️ 编译说明

本项目使用 **MinGW (g++)** 编译，在项目根目录下执行以下命令：

### 编译菜单程序
```bash
g++ -o menu.exe menu.cpp
```

### 编译贪吃蛇
```bash
g++ -o hello.exe hello.cpp
```

### 编译飞机大战
```bash
g++ -o airplane_war.exe airplane_war.cpp
```

### 一键编译所有程序
```bash
g++ -o menu.exe menu.cpp && g++ -o hello.exe hello.cpp && g++ -o airplane_war.exe airplane_war.cpp
```

编译环境要求：
- MinGW-w64（GCC 8.1.0 或更高版本）
- Windows 系统（使用了 `<windows.h>` 和 `<conio.h>` 库）
